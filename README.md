# Wordpress
Repository to structure Wordpress projects

## Setup with Docker
## Most need
```shell
brew install make
brew install docker
```

## First run
```shell
make remove
make config
make install
make init
```

## Dev mode
```shell
make init
```

### Access
Http://localhost:5000
